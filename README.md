# SimCity – Algorithms and Data Structures II

This main project is based on a simplified version of the popular life simulator game, focusing on the 'Abstract Data Types' specification and its implementation in C++23.

Players can:
* Add a home
* Add a shop
* Merge two maps into one.
The map is defined at the start of the game, and the goal is to get as many points as possible, using a combination of building positioning and turns passed.

## Contents
* In SimCity_Part1.pdf you will find the ADT formal specification.
* At the SimCity_Part2 directory you'll encounter the implementation using C++23.

### Compilation and Testing
To compile the project:
1. `mkdir build`
2. `cd build`
3. `cmake ..`
4. `make`

From the project SimCity_Part2 directory:
1. `cd build/`
2. `./correrTests`
